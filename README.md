# Unified Event Analytics Engine

A backend service for collecting analytics events (clicks, page views, device info, etc.) from web and mobile apps.

## Tech Stack
- Node.js
- TypeScript
- Express.js
- PostgreSQL (Railway)
- Prisma ORM
- Redis (Railway)
- Swagger UI
- Docker

## Run Locally
npm install
npm run dev
